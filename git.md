# Подсказак по Git

Создали репозиторий 
```
git init
```
Добавление отдельных файлов или всех файлов в область подготовленных файлов
```
git add somefile.js
```
 Внесение изменений однострочным сообщением или через редактор
 ```
 git commit - m 'комиты'
 ```
  Просмотр истории коммитов с изменениями
  ```
  git log
  ```
  в одну строку 
  ```
  git log --p
  ```
